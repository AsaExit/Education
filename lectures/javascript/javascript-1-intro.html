What is JavaScript (JS)?

  • JavaScript is the programming language of HTML and the Web.
  • JavaScript and Java are completely different languages.
  • JavaScript is a scripting language mostly used on the web
  • JavaScript code is written into an HTML page and interpreted by the browser
  • Can be used outside the browser as well (node.js, Apache CouchDB)
  • ECMAScript 2015 (ES6)= Standard for JavaScript

Why JavaScript?

  • JavaScript adds interactivity to your website
    • Response to user actions (clicking buttons etc.)
    • Changing the site dynamicly like Styling
    • Animation
