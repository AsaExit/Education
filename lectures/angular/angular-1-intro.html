Building blocks

Template

  • A template combines HTML with Angular markup -> modifies HTML elements before being displayed

Services

  • Typically a class
  • Logic not related to a specific view
  • Can be injected into a Component
  • Example fetch and handle data, store user
  • Documentation

Modules

  • Bundles functionalities of our app
  • A group of related components
  • Angular provides their functionality in modules
  • Example - Admin, Users, Authentication
  • Documentation
