HTML

Hypertext Markup Language

What is HTML?

  • HTML is a markup language
  • It describes the structure of a Web pages (marks it)
  • HTML Elements are the building blocks of HTML pages
  • These elements are represented by "tags"
  • The tags are label the pieces of content
  • Browsers do not display the tags, but uses them to render the content
  • index.html

Browsers

  • Chrome
  • Firefox
  • Safari
  • Edge
  • Internet Explorer
  • These read HTML documents and display them (do not show the tags, but uses them to display the document).
